# OpenVPN-Access-Management
 * Note: This project was developed to provide a more convenient and professional solution for managing VPN users
## 1. Giao diện Dashboard
  Giao diện hiển thị thông tin về Server VPN, thông tin về client đang có trong hệ thống:
  + Clients Total,
  + Clients Disconected,
  + Clients Revoked,
  + Clients Running,
  + Peichart,
  + Ip used

## 2. Giao diện Client
  Ở giao diện này cho phép:
  
  + Tìm kiếm Client,
  + Lọc Client,
  + Tạo mới Client,
  + Xóa Client hiện tại,
  + Revoke Client hiện tại,
  + Download profile VPN,
  + Xem thông tin chi tiết Client
